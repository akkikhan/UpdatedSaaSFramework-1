SDK Integration Guide

NPM packages for seamless authentication and RBAC integration

Authentication SDK

Installation

npm install @saas-framework/auth

Basic Usage

import { SaaSAuth } from '@saas-framework/auth';

const auth = new SaaSAuth({
  apiKey: 'auth_your-tenant-key',
  baseUrl: 'https://api.com/api/v2/auth'
});

// Login user
const session = await auth.login(
  'user@tenant.com', 
  'password'
);

// Verify token
const isValid = await auth.verifyToken(token);
JWT token management
Express middleware support
User session management

RBAC SDK

Installation

npm install @saas-framework/rbac

Basic Usage

import { SaaSRBAC } from '@saas-framework/rbac';

const rbac = new SaaSRBAC({
  apiKey: 'rbac_your-tenant-key',
  baseUrl: 'https://api.com/api/v2/rbac'
});

// Check permission
const canCreate = await rbac.hasPermission(
  userId, 
  'user.create'
);

// Middleware protection
app.use(rbac.middleware(['user.read']));
Role-based access control
Permission checking
Express route protection

Combined Usage Example 

// Protected route with authentication and authorization
app.get('/users',
  auth.middleware(),                    // Validate JWT
  rbac.middleware(['user.read']),       // Check permission
  async (req, res) => {
    // Route logic - user is authenticated and authorized
    res.json({ users: await getUsers() });
  }
);
